# Nikkei 225 Futures Signal App

日経平均フューチャーズ向けの売買判断支援Webアプリです。

## 内容

- TradingView の Nikkei 225 Futures 60分足チャート
- 現在値、直近高値、直近安値、上値抵抗、下値支持の入力
- 買い検討、売り検討、見送りゾーンの自動表示
- ローソク足、移動平均、フィボナッチ、価格帯別アクション
- リスク許容額から見た枚数目安

## 使い方

`index.html` をブラウザで開くだけで使えます。

TradingView の表示はデータ提供条件により、リアルタイムまたは遅延になる場合があります。

## 注意

このアプリは売買判断の補助ツールです。投資助言ではありません。
実際の注文前に、現在値、板、出来高、ニュース、為替、米国株、損切り幅を必ず確認してください。
